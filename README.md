### The Most Beautiful & Functional Grocery App Built with Pure HTML, CSS & JavaScript

SmartList Pro is a next-generation grocery management web application that combines stunning design with powerful functionality — all without any framework.

---

### Features

| Feature                      | Description                                                                                   |
|-----------------------------|------------------------------------------------------------------------------------------------|
| Voice Recognition           | Just speak — "3 apples and 2 liters of milk" → instantly added with correct quantity         |
| Real-Time Budget Tracking   | See total cost update live. Know exactly how much you're spending                           |
| Smart Auto-Categorization   | AI automatically sorts items into Fruits, Vegetables, Dairy, Meat, Drinks, etc.              |
| Priority Management         | Mark critical items — they stay at the top with special highlighting                         |
| Quantity & Price Support    | Add quantity and price for each item → perfect budget control                                 |
| Dual Pro Themes             | • Sunny Yellow-Blue (Day mode) <br> • Neon Red Xenon Cyberpunk (Night mode)                  |
| Offline-First & Persistent  | Works 100% offline. Your list is saved forever using LocalStorage                             |
| Fully Responsive            | Looks stunning on mobile, tablet, and desktop                                                 |

---

### Tech Stack

- HTML5 – Semantic & accessible structure
- CSS3 – Modern Glassmorphism, Gradients, Animations, Custom Properties
- Vanilla JavaScript (ES6+) – No frameworks, no dependencies
- Web Speech API – Native voice recognition
- LocalStorage – Persistent data
- Font Awesome 6 – Beautiful icons
- Google Fonts (Inter) – Professional typography
